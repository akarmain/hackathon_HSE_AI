Контекст проекта:
- Монорепозиторий: backend (FastAPI) + frontend (Nuxt)

- Система генерирует сценарий, потом слайды по одному, показывает прогресс и дает скачивание
- Есть uploads, genAI (text + image), orchestrator, HTML->PNG->PDF рендер

Стиль:
- Для защиты: понятно, кратко, структурно
- 8–10 слайдов
- Тон: уверенный, но без перегруза
- Показывать результат по шагам

Требования к изображениям:
- Обязательно использовать изображения, загруженные пользователем, через ключи:
- logo_team (логотип команды) — использовать на титульном и финальном слайде
- product_ui (скрин интерфейса) — слайд “Демо”
- architecture_diagram (схема) — слайд “Архитектура”
- process_flow (блок-схема) — слайд “Пайплайн генерации”

1) Титул (название проекта + команда + логотип logo_team)
