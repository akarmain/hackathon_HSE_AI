Сделай презентацию для защиты хакатон-проекта «AI генератор презентаций».

Контекст проекта:
- Система генерирует готовые слайды по теме пользователя.
- Архитектура: Master-агент (сценарий, структура, тексты, промпты для изображений) + Worker-агент (детерминированная генерация HTML слайда по инструкции).
- Поддержка выбора моделей для сценария и кода через константы.
- Поддержка загрузки пользовательских фото с ключами (описаниями), логическое распределение фото по слайдам, включая дублирование на нужных слайдах.
- Параллельная генерация слайдов с ограничением concurrency.
- Просмотр слайдов в полноэкранном режиме, поэтапная подгрузка, экспорт в PDF.
